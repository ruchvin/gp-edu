# gp-edu
Занятие 4. Развёртывание и базовая конфигурация: Multi-node // ДЗ
Домашнее задание
Вариант 1: установка Single-Node версии в докере.
Вариант 2: установка Single-Node версии на виртуальной машине.
Вариант 3: установка Multi-Node версии на нескольких виртуальных машинках.
Для успешной сдачи домашней работы необходимо выполнение одного из указанных вариантов.
Формат сдачи: ссылка на Git репозиторий с файлами скриптов и файлом readme.

Создаём папку на сервере, где установлен docker
mkdir /opt/greenplum-single-node-docker

загружаем в неё файл docker-compose.yml

переходим в папку 
cd /opt/greenplum-single-node-docker

запускаем контейнер
docker-compose up -d

подключаемся 
# Хост: <IP адрес сервера>
# Порт: 7432 
# База: testdb
# Пользователь: gpadmin
# Пароль: gpadmin

Используется порт 7432 так как в моём случае стандартный порт 5432 занят.
