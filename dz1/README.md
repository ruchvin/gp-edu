Для успешной сдачи домашней работы необходимо выполнение одного из указанных вариантов.
Формат сдачи: ссылка на Git репозиторий с файлами скриптов и файлом readme.
### Выполнение
0. Конфигурация сервера 
 - ОС:  Ubuntu 20.04.6
 - Установлен Docker

1. Создаём папку на сервере
```mkdir /opt/greenplum-single-node-docker```

2. Загружаем в неё файл docker-compose.yml

3. Переходим в папку 
```cd /opt/greenplum-single-node-docker```

4. Запускаем контейнер
```docker-compose up -d```

5. Подключаемся 
- Хост: <IP адрес сервера>
- Порт: 7432 
- База: testdb
- Пользователь: gpadmin
- Пароль: gpadmin
> [!NOTE]
> Используется порт 7432, sтак как в моём случае стандартный порт 5432 занят.
